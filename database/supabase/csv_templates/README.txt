Supabase CSV templates for SmartHub Portal

IMPORTANT:
- CSV import in Supabase imports data rows only.
- CSV cannot create tables automatically.
- First run schema SQL:
  website/database/supabase/smarthub_portal_schema.sql

Then import each CSV file into its matching table in Supabase Table Editor.

These template files currently contain only header rows.
Add your data rows below the header before importing.
